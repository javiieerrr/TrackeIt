# TrackIt 🎯

Aplicación desarrollada en JavaScript para gestionar objetivos personales.

### Funcionalidades

* Agregar objetivos.
* Asignar dificultad (1–5).
* Validar entradas del usuario.
* Marcar objetivos como completados.
* Eliminar objetivos.

### Tecnologías

* HTML
* CSS
* JavaScript (ES6)

### Estructura

* `index.html` → interfaz.
* `styles.css` → estilos.
* `logica.js` → objetos y lógica.
* `app.js` → eventos y DOM.

### Conceptos aplicados

* Variables y condicionales.
* Funciones.
* Arreglos y ciclos (`forEach`).
* Objetos y clases.
* Validación de datos.

### Uso

Ingresa un objetivo, selecciona dificultad y agrégalo. Luego puedes completarlo o eliminarlo.

### Mejoras futuras

* Editar objetivos existentes antes de completarlos.
* Mostrar estadísticas de progreso.
* Guardar objetivos usando LocalStorage.
* Filtrar objetivos completados y pendientes.
